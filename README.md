# Gomit

Gomit [Golang Committer] | Beauty git commit using Go.

```
--[ gomit ]--

✨ : new      📝 : doc       ⏫ : up
🛠 : fix      🔀 : merge     📦 : package
🔨 : update   ⏬ : down      🚧 : working

Usage :
./gomit -a=up -m='Create some function to file'
```

## Installation

```shell
$ go build gomit.go
$ ./gomit -a=up -m='Create some function to file'
```

#### Add to /usr/bin

```shell
$ sudo sh install.sh
```
